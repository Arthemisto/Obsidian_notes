## 🔗 Links: Hard link vs Symbolic link (soft link)
**Коротко:** в Linux есть два типа ссылок: **hard link** (ещё одно имя тех же данных) и **symbolic link / symlink** (отдельный файл-указатель на путь).
![[Pasted image 20260304170842.png|552]]

---
### 🧱 Hard link
- По сути: **другое имя того же файла** (те же данные на диске).
- Оригинал и hard link **указывают на одно и то же**.
- Удалил “оригинал” → данные **остаются**, пока существует хотя бы один hard link.
- ❌ Нельзя создавать для **директорий**.
- ❌ Нельзя делать между **разными файловыми системами/разделами**.
✅ Команда:
ln source.txt hardlink.txt

---
### 🪢 Symbolic link (symlink / soft link)
- По сути: **отдельный файл**, который хранит **path** к цели (файлу или директории).
- Если удалить цель → symlink станет **broken/dangling**.
- ✅ Можно между **разными файловыми системами/разделами**.
- ✅ Можно создавать на **директории**.
✅ Команда:
ln -s /path/to/target symlink_name
