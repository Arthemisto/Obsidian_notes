**Коротко:** `grep` — утилита командной строки в Unix/Unix-like системах для поиска **текстовых шаблонов** в файлах. Можно искать по обычной строке или по **regular expressions ([[regex]])** и применять поиск к одному или множеству файлов.

---
## 🧩 Что делает grep
- Ищет **pattern** (шаблон) в тексте.
- Работает:
  - с одним файлом
  - с группой файлов (например `*.txt`)
- Поддерживает **regex** для более гибкого поиска.
---
## ✅ Примеры (EN)  
  
| команда | значение |  
|---|---|  
| `grep "string" file_name.txt` | найти `string` в файле |  
| `grep -i "string" file_name.txt` | поиск без учёта регистра (ignore case) |  
| `grep -n "string" file_name.txt` | вывод совпадений с номерами строк |  
| `grep "string" *.txt` | поиск по всем `.txt` файлам в папке |  
| `grep "str.*ing" file_name.txt` | regex: между `str` и `ing` может быть любой текст |  
| `grep "^string" file_name.txt` | совпадение только в начале строки |  
| `grep "string$" file_name.txt` | совпадение только в конце строки |  
  
---
## 🧠 Мини-памятка по regex якорям
- `^` — начало строки  
- `$` — конец строки  
---
## 🧭 Summary / Сводка
`grep` — базовый инструмент поиска в Linux: быстро находит нужные строки, поддерживает regex и легко масштабируется на много файлов (`*.txt`).
