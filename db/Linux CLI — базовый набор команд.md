# Linux CLI — базовые команды
#os #linux #cli #terminal #apt #sudo
**Коротко:** эти команды — “минимальный набор выживания” в терминале Linux: навигация, создание/удаление файлов, копирование/перемещение, админ-права и установка софта.

---
## 🧰 Core commands (must-know)

| команда | значение |
|---|---|
| `ls` | показать содержимое директории (файлы/папки) |
| `cd` | перейти в директорию |
| `pwd` | показать текущий путь (где ты находишься) |
| `mkdir` | создать директорию |
| `touch` | создать пустой файл (или обновить timestamp) |
| `cp` | копировать файлы/папки |
| `mv` | переместить или переименовать файл/папку |
| `rm` | удалить файл/папку |
| `sudo` | выполнить команду с правами суперпользователя |
| `apt` | менеджер пакетов (установка/обновление/удаление ПО) |

---

## ⚡ Частые варианты (добавлено для практики)

| команда | значение |
|---|---|
| `ls -l` | “длинный формат”: права, владелец, размер, дата |
| `ls -a` | показать скрытые файлы (начинаются с `.`) |
| `cd ..` | перейти на уровень выше |
| `cd ~` | перейти в домашнюю папку |
| `mkdir -p a/b/c` | создать цепочку папок сразу |
| `cp -r dir1 dir2` | копировать папку рекурсивно |
| `mv old new` | переименовать (если `new` в той же папке) |
| `rm -r folder` | удалить папку рекурсивно |
| `rm -f file` | удалить без вопросов (осторожно) |
| `sudo apt update` | обновить список пакетов |
| `sudo apt upgrade` | обновить установленные пакеты |
| `apt search name` | поиск пакета по имени |
| `sudo apt install name` | установить пакет |
| `sudo apt remove name` | удалить пакет (без конфигов) |
| `sudo apt purge name` | удалить пакет + конфиги |
| `apt show name` | информация о пакете |

## 🪟 `dir` vs 🐧 `ls`
- **`ls`** — стандартная команда Unix/Linux для просмотра содержимого директории.
- **`dir`** — команда Windows CMD; в Linux часто тоже доступна, но обычно это просто разновидность `ls` (из coreutils).
- **Правило:** в Linux используем **`ls`**, `dir` — только если привычка/сравнение с Windows.
## 🧠 Мини-правило безопасности
- **`sudo`** и особенно **`rm -rf`** — это “режим без страховки”. Всегда дважды проверяй путь и имя.
